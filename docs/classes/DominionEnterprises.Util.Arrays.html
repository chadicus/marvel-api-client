project(array $input, mixed $key, boolean $strictKeyCheck = true) : array
static

if $input = array(

array('key 1' => 'item 1 value 1', 'key 2' => 'item 1 value 2'),
array('key 1' => 'item 2 value 1', 'key 2' => 'item 2 value 2'),
array('key 1' => 'item 3 value 1'),

) and $key = 'key 2' and $strictKeyCheck = false

then return array('item 1 value 2', 'item 2 value 2')

but if $strictKeyCheck = true then an InvalidArgumentException occurs since 'key 2' wasnt in item 3

if a value in $input was not an array

if a key was not in one of the $input arrays

$key

the key which values we are to project

ensure key is in each $input array or not

the projection

Takes each item and embeds it into the destination array, returning the result.

embedInto(array $items, string $fieldName, array $destination = array(), bool $overwrite = false) : array
static

Each item's key is used as the key in the destination array so that keys are preserved. Each resulting item in the destination will be embedded into a field named by $fieldName. Any items that don't have an entry in destination already will be added, not skipped.

For example, embedInto(['Joe', 'Sue'], 'lastName', [['firstName' => 'Billy'], ['firstName' => 'Bobby']]) will return [['firstName' => 'Billy', 'lastName' => 'Joe'], ['firstName' => 'Bobby', 'lastName' => 'Sue']]

throws

if $fieldName was not a string

if a value in $destination was not an array

if $fieldName key already exists in a $destination array

Arguments

$items

array

The items to embed into the result.

$fieldName

string

The name of the field to embed the items into. This field must not exist in the destination items already.

$destination

array

An optional array of arrays to embed the items into. If this is not provided then empty records are assumed

and the new record will be created only containing $fieldName.

$overwrite

bool

whether to overwrite $fieldName in $destination array

Response

array

$destination, with all items in $items added using their keys, but underneath a nested $fieldName key.
